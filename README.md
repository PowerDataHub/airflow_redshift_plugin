# Airflow Plugin - Redshift

Move data from [Amazon Redshift](https://aws.amazon.com/pt/redshift/) to 3rd party sources

## Operators

### RedshiftToDruidOperator

Unloads a Redshift table to S3 and load into Apache Druid
